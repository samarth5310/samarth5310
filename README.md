## ◈ About

I am a final-year Computer Science and Engineering student at BGMIT, Mudhol, with a strong foundation in full-stack software engineering and an emerging focus on intelligent systems. I design and build production-grade web applications — from real-time indoor navigation platforms using sensor fusion and pathfinding algorithms, to admin-driven form builders and data visualization dashboards.

My engineering philosophy is rooted in clean architecture, performance at scale, and product thinking. I approach every system — frontend or backend — with the same discipline: modular design, testable logic, and measurable outcomes.

On the AI/ML frontier, I apply computer vision, dead reckoning, SLAM-lite techniques, and graph-based pathfinding (A\*, Dijkstra) to practical problems. I build systems that are not just functional but intelligent — capable of adapting to dynamic real-world conditions.

Beyond code, I drive initiatives: organizing department-wide hackathons, mentoring peers, and translating engineering complexity into impact that non-technical stakeholders can recognize and trust.

**Open To:** Full-time SWE roles · AI/ML engineering · Full-stack product engineering · Research collaborations · Open-source partnerships

## ◈ Featured Projects

<details>
<summary><strong>◉ Indoor Navigation System — Browser-Native Intelligent Wayfinding</strong></summary>

<br/>

A production-grade, browser-based indoor navigation platform designed for GPS-denied environments. The system fuses QR-code localization, dead reckoning via device IMU, and a SLAM-lite mapping engine to deliver real-time wayfinding without native app installation. Pathfinding is powered by dual A\* and Dijkstra algorithm engines with dynamic obstacle avoidance and accessibility route preferences.

| Attribute | Details |
|-----------|---------|
| **Stack** | React · Vite · TypeScript · Flask · Python · Firebase · OpenCV |
| **Architecture** | Multi-phase modular build · REST API · WebSocket real-time updates |
| **Algorithms** | A\* · Dijkstra · Dead Reckoning · SLAM-lite · QR Anchor Localization |
| **Performance** | Sub-200ms pathfinding on dense grid graphs · Mobile-optimized rendering |
| **Security** | Role-based access control · Auth-gated admin dashboard |
| **Impact** | Capstone project · Peer-reviewed academic documentation |
